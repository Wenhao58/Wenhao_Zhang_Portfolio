📊 Tableau Sales Performance Dashboard
Project Overview
This project showcases an interactive Tableau dashboard built to visualize key sales metrics from the AdventureWorks dataset. The dashboard highlights product performance, revenue trends, and customer purchasing behavior through dynamic charts and filters.
It demonstrates your ability to transform raw data into clear, actionable insights using modern BI tools.

Objectives
- Create an interactive dashboard for exploring AdventureWorks sales data.

- Visualize product-level performance using bar charts, KPIs, and filters.

- Display monthly revenue trends using line charts.

- Provide drill‑down capabilities for deeper analysis (product category, subcategory, individual products).

- Present insights in a clean, professional layout suitable for business stakeholders.

Tools & Technologies
- Tableau Desktop

- AdventureWorks2016 dataset

- SQL Server (data extraction)

- Excel / CSV (optional intermediate storage)

Key Skills Demonstrated
- Business intelligence dashboard design

- Data modeling and cleaning for visualization

- KPI creation (total revenue, total units sold, average order value)

- Interactive filters and parameters

- Trend analysis and storytelling with data

- Professional layout and UX design in Tableau

Dashboard Features
1. Sales Overview
   Total revenue

   Total units sold

   Average order value

   Year‑over‑year comparison (if applicable)

2. Product Performance
   Bar charts showing top‑selling products

   Category and subcategory filters

   Highlight actions for drilling into specific items

3. Monthly Sales Trends
   Line chart visualizing monthly revenue

   Seasonal patterns and growth trends

   Tooltip details for each month

4. Customer Insights (optional)
   Customer segments

   Order frequency

   Revenue contribution by customer type

Workflow
- Extract data from SQL Server using queries.

- Clean and prepare data in Excel or directly in Tableau.

- Build visualizations (bar charts, line charts, KPIs).

- Combine views into a cohesive dashboard layout.

- Add interactivity (filters, actions, tooltips).

- Publish dashboard for stakeholder review.

Future Improvements
- Add forecasting using Tableau’s built‑in trend models.

- Include customer segmentation using clustering.

- Build a second dashboard focused on inventory or operations.

- Publish the dashboard to Tableau Public for public viewing.

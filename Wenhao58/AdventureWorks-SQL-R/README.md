📊 AdventureWorks Sales Analysis (SQL + R)
Project Overview
This project analyzes 2013 sales data from the AdventureWorks2016 SQL Server database using SQL for data retrieval and R for data manipulation, summary statistics, visualization, and predictive modeling.
The work was completed as part of a multi‑component analytics project involving discovery analytics, cumulative statistics, and linear regression forecasting.

Objectives
Identify Top 10 products by:

Unit Volume

Sales Revenue

Generate summary statistics for all 2013 sales orders.

Create visualizations:

Pie charts

Bar charts

Monthly sales line charts for top products

Build linear regression models to forecast 2014 sales for the top 3 revenue‑generating products.
